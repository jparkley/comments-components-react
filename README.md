# React Components to Display User Comments.

These components display user comments with name, date and content information.  At the bottom of each comment card, there are two buttons, "Approve" and "Deny".

## Features
- Design from semantic-ui
- Images from Faker

```sh
Design from semantic-ui
Faker module: npm install --save faker
```
